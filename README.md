# Actividad-5


## Cómo ejecutar el proyecto
1. Descarga los archivos `actividad_hola_mundo.html` y `actividad.js` en una misma carpeta.
2. Hacer doble clic en el archivo `actividad_hola_mundo.html` para abrirlo en cualquier navegador web (Edge, Chrome, Firefox).
3. Al abrir, se vera la alerta emergente.
4. En la página principal se veran los mensajes inyectados en el HTML.
5. Para verificar la salida por consola, presiona `F12` (o haz clic derecho y selecciona "Inspeccionar") y abre la pestaña "Consola".
